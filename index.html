Категория Cephalon

Основные команды

  • help - Справка по командам
  • botver - Версия бота
  • devs - Разработчики
  • info - Информация о боте
  • invite - Пригласить бота
  • locale - Настройки языка
  • ping - Проверка задержки
  • uptime - Время работы бота

Категория Embeds

Работа с сообщениями

  • content - Показать контент сообщения
  • edit - Редактировать сообщение
  • say - Отправить сообщение через бота

Категория Fun

Развлекательные команды

  • aghanim - Проверка длины своего аганима
  • dotersbrain - Проверка на мозг дотера
  • roulette - Русская рулетка
  • leaderboard - Таблица лидеров по победам
  • settings - Настройки рулетки

Категория Mod

Модерация

  • ban - Забанить пользователя
  • clear - Очистить сообщения
  • dm - Отправить ЛС от имени бота
  • deaf - Заглушить в голосовом канале
  • give - Выдать роль
  • kick - Кикнуть пользователя
  • mute - Заглушить в текстовых каналах
  • take - Забрать роль
  • timeout - Таймаут пользователя
  • unmute - Снять заглушение

Категория Misc

Разное

  • about - Информация о пользователе
  • avatar - Аватар пользователя
  • coinflip - Подбросить монетку
  • roll - Случайное число
  • roleinfo - Информация о роли
  • rolemembers - Участники с ролью
  • serverinfo - Информация о сервере
  • someone - Случайный участник
